How it works

01

Start Recording

> brief --start
Starting terminal session recording...

Start recording your session before you tackle the box. Export your HUGGING FACE TOKEN first.

02

Solve the Box

> nmap -sC -sV 10.x.x.x
... solving the machine normally ...

Work through the machine as you usually would. Brief silently records your workflow in the background.

03

Generate Report

> brief --stop
> brief --latest

Stop the recording and generate a structured AI-powered after-action report detailing your methodology.

CLI Interface

Core Commands

brief --start terminal

Starts recording the current terminal session.

brief --stop terminal

Stops the recording process and saves logs.

brief --latest terminal

Generates an AI report based on the most recent session.

brief -i <file> terminal

Generates a report for a specific saved session log.
