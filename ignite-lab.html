Core Vulnerabilities

code

Remote Code Execution (RCE)

Exploitation of CVE-2018-16763 in Fuel CMS 1.4.1. The vulnerability exists in the `pages/select` endpoint, allowing for arbitrary PHP code execution via a crafted GET request.

CVE-2018-16763 PHP Deserialization Web Shell

Default Credentials

Many services on this machine utilize default installation credentials, highlighting the importance of proper hardening post-deployment.
